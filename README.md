# Mastermind

## Requirements
* Ruby 2.3.3 (or later)
* RSpec 3.7 (or later)
* For best compatibility use MacOS (High Sierra or later) or Ubuntu (18.04 or later)

## Running Mastermind
1. run `ruby lib/mm.rb`

## Running Tests
1. If rspec is not installed, do so by running `gem install rspec`
2. run `rspec`
